### Tailwind css configuration

- Add darkMode property

```ts title="tailwind.config.{ts, js}" {2, 7-9}
export default {
+  darkMode:"selector",
  theme: {
    extend: {
      // ...
    },
+    boxShadow: {
+      "inner-2xl": "inset 10px 10px 10px -1px rgba(10,99,169,0.15), inset -10px -10px 10px -1px rgba(255,255,255,0.70)",
+    },
  },
  plugins: [],
}
```
